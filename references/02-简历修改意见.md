# 02-简历修改意见

用于阶段 2 和阶段 3：审查简历匹配度、找问题、做能力迁移和简历改写。

## 核心诊断

很多转岗简历的核心问题不是“没有能力”，而是“原岗位自证”：设计师只证明自己会设计，运营只证明自己做活动，技术只证明自己写代码。AI 产品经理简历必须把原经历翻译成招聘方能识别的语言：

```text
用户/业务问题 -> 需求拆解 -> 方案设计 -> AI/工具链落地 -> 协作推进 -> 指标/反馈 -> 复盘迭代
```

## 常见问题与修改方向

| 问题 | 表现 | 修改方向 |
|---|---|---|
| 头部定位自降身价 | 写“转行/转向 AI 产品” | 直接写目标身份，如“AI 产品助理 / AIGC 产品专员 / AI Workflow 产品方向” |
| 行业黑话过多 | 景观/设计/运营术语堆满，HR 无法对应 | 翻译成需求拆解、流程管控、SOP、跨团队协作、交付闭环 |
| AI 项目像兴趣作品 | 只展示生成图、网站、工具使用 | 改成从 0 到 1 的 AI 工作流/产品化项目 |
| ATS 关键词不足 | 没有 RAG、Agent、Prompt、Workflow、评测等 | 在摘要、技能、项目 bullet 中自然分布 |
| 成果无量化 | 全是“负责/参与/优化” | 补项目数量、周期、效率、交付率、产出规模；不能编造 |
| 作品集偏设计 | 网站首屏像景观作品展示 | 改成 AI 产品案例页，先展示能力和项目闭环 |

## 简历首屏建议

不要写：

```text
[年限] 年 [原岗位/行业] 经验，转向 AI 产品经理
```

可以写：

```text
AI 产品助理 / AIGC 产品专员 / AI Workflow 产品方向
[年限] 年复杂项目全流程推进经验，具备 AI 工具链实践、需求拆解、信息架构、流程标准化与跨团队协作能力。
```

如果投保底线：

```text
数字化业务运营 / AI 工具流程协调
[年限] 年项目流程管控经验，具备 SOP 搭建、跨团队协同、AI 工具落地与英文文档处理能力。
```

## 能力迁移表

| 原经历 | 可迁移能力 | AI 产品表达 |
|---|---|---|
| 景观/建筑方案设计 | 多约束需求理解、用户场景判断、方案取舍 | 能将复杂用户需求拆成产品流程和功能模块 |
| 项目全流程管理 | 节点推进、跨角色协同、交付标准化 | 能推动需求从调研、原型、开发到验收 |
| 作品集/网站搭建 | 信息架构、内容组织、交互表达 | 能完成从 0 到 1 的产品原型和产品叙事 |
| AI 生成图/Prompt 实践 | AIGC 工作流、提示词模板、结果筛选 | 能构建生成-筛选-校验-反馈闭环 |
| 英文读写 | 海外产品研究、英文文档、跨国协作 | 能阅读 AI 产品文档并支持外企/跨境场景 |

## 项目改写模板

```markdown
### [项目名]：AI 设计/内容生成 Workflow Demo
- 面向 [目标用户]，发现 [核心痛点]，设计 [AI 工作流/产品方案]。
- 拆解 [输入信息、任务节点、输出物、校验标准]，沉淀 [Prompt 模板/流程 SOP/评估维度]。
- 使用 [工具/API/平台] 完成 [原型/网站/Demo/流程验证]，覆盖 [场景数量/功能模块]。
- 通过 [用户反馈/效率对比/bad case 复盘] 验证效果，后续迭代方向为 [优化点]。
```

## 推荐关键词

AI 产品主线：

```text
AI Product, AIGC, LLM, Prompt Engineering, AI Workflow, Agent, RAG,
多模态内容生成, 用户意图识别, 任务拆解, 信息架构, 产品原型,
生成-筛选-校验-反馈闭环, 内容质量评估, 模型评测, bad case 归因
```

数字化运营保底线：

```text
SOP 搭建, 流程管控, 跨职能协同, 文档标准化, 业务运营,
数字化提效, 内部工具落地, 用户反馈收集, 项目协调, 英文文档
```

## 个人网站/作品集改造

首屏优先展示：

```text
AI 产品方向定位
核心能力标签
下载简历
查看 AI 产品案例
```

导航建议：

```text
首页 -> AI 产品案例 -> 核心能力 -> 项目管控案例 -> 关于我
```

AI 项目页必须包含：

- 目标用户与痛点
- 产品流程图或工作流图
- Prompt/Agent/RAG/工具链说明
- 输出结果示例
- 质量评估标准
- bad case 与迭代
- 可迁移到 JD 的能力说明

## 简历版本

至少准备两版：

- `AI 产品主线版`：投 AI/AIGC 产品、AI Workflow、AI 产品运营、设计/内容工具。
- `数字化运营保底版`：投外企流程运营、内部 AI 工具推广、产品支持、项目协调。
